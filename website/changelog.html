Changelog

Notable changes to Kaeda across releases.

0.1.0 TBD
First public release
  • Core mining workflow — Load video + SRT, navigate subtitles, select words, save cards to a session.
  • Korean tokenization — Click any word to select it as the card target; automatic morphological analysis.
  • Dictionary lookup — Auto-filled explanations from built-in Korean dictionary.
  • Card management — Create, edit, and delete cards per session.
  • Deck system — Multiple decks with one active session.
  • TSV export — One-click Anki-ready export.
  • Known-line tracking — Persisted per file; dimmed in future sessions.
  • Video player — Custom controls: play/pause, seek, speed, volume, mute.
  • Subtitle search — Full-text search with keyboard navigation.
  • Cross-platform builds — macOS (ARM + Intel), Windows, Linux.
