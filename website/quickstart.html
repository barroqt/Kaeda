Quick Start

From video to Anki in minutes.

1. Install Kaeda

macOS — Download the archive for your chip (Apple Silicon or Intel), extract, and drag Kaeda.app to Applications.

Windows — Download the archive, extract, and run Kaeda.exe.

Linux — Download the archive, extract, and run ./kaeda.

2. Open a video and subtitle file

  1. Click Open Video and select an .mp4 or .mkv file.
  2. Click Open Subtitles and select the matching .srt file.
  3. The subtitle list appears on the left. The video appears in the center.
If your video has embedded subtitles, use File → Open Video with Embedded Subtitles instead.

3. Navigate subtitles and select lines

Use the keyboard or mouse to move through subtitles:

ShortcutAction
W / SPrevious / next subtitle
A / DPrevious / next word
SpacePlay / pause video
KMark the current line as known
RReplay current subtitle audio

Clicking a subtitle in the list seeks the video to that line and makes it the current selection.

4. Create cards

  1. Navigate to a subtitle you want to mine.
  2. Click a word in the sentence to select it as the target.
  3. A dictionary lookup runs automatically — the definition appears in the card preview panel.
  4. Edit the explanation or add notes if desired.
  5. Press Cmd+Enter or click Save Card.

5. Export your deck

Click Export and choose a location. A .tsv file is written with three columns: Target, Sentence, Explanation.

6. Import into Anki

  1. Open Anki and go to File → Import.
  2. Select the exported .tsv file.
  3. Set the field mapping: Field 1 → Target, Field 2 → Sentence, Field 3 → Explanation.
  4. Set the deck and note type, then click Import.

Tips

Auto-translation — Set up a DeepL API key in Settings to get automatic translations for each subtitle span.

Known lines — Marking a line as known dims it in the subtitle list and hides it in future sessions for the same file.

Multiple decks — Create and switch between decks from the deck selector in the toolbar.
