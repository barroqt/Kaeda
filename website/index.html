The Problem

Current mining workflows are broken.

Broken Immersion

Pause video → copy text → open translation app → copy translation → open Anki → assemble card → paste translation. Repeat 50 times.

Language Mismatch

Generic tools fail to handle Korean's complex morphology, leading to messy cards.

Friction Overload

The friction is so high that most learners give up on active mining entirely, even though they know it works.

Core Features

Everything you need in one place.

📂

Video + SRT in seconds

Load any local MP4 and SRT file instantly. No complex setup, just your content.

⌨️

Keyboard workflow

You have the option to do an entire mining session without touching you mouse.

🏮

In-app translation

Click any word and get definition auto-filled. Edit freely before saving.

🔊

Audio replay

Replay the exact audio clip for any subtitle line.

🧠

Known-word tracking

Mark lines as Known and Kaeda remembers them. Future sessions for the same file skip what you've already learned.

📤

Anki TSV Export

One-click export of your entire session. Drag the file into Anki and your new cards are ready.
